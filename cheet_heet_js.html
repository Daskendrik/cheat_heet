Методы чисел

  1. Если много нулей можно использовать e - 123e6 = 123000000
  2. Math.floor - округление в меньшую сторону
  3. Math.ceil - округление в большую сторону
  4. Math.round - округление до ближайшего целого
  5. Math.random() - возвращает рандомное число от 0 до 1
  6. Math.max(a, b, c...) / Math.min(a, b, c...) - возвращает максимум или минимум
  7. Math.pow(n, power) - Возвращает число n, возведённое в степень power
  8. Math.trunc - производит удаление дробной части без округления: 3.1 становится 3
  9. num.toFixed(n) - округляет число до n
  10. isNaN(value) преобразует значение в число и проверяет является ли оно NaN
  11. isFinite(value) преобразует аргумент в число и возвращает true, если оно является обычным числом, т.е. не NaN/Infinity/-Infinity
  12. Number() или +str - преобразование в число
  13. parseInt()/parseFloat() - вытаскивает из строки число и преобразует в число, то только, если число идет первым в строке, иначе NaN

Методы строк

  1. Писать строчку можно и в `` - тогда можно через ${} вставлять переменные
  2. Спецсимволы:
    • \n Перевод строки
    • \r В текстовых файлах Windows для перевода строки используется комбинация символов \r\n
    • \', \" Кавычки
    • \\ Обратный слеш
    • \t Знак табуляции
    • \b, \f, \v Backspace, Form Feed и Vertical Tab — оставлены для обратной совместимости, сейчас не используются.
  3. str.length - длина строки
  4. str[0] - доступ к первому символу
  5. str.toLowerCase() / str.toUpperCase() - все символы строчные/заглавные
  6. str.indexOf(substr, pos) - поиск substr в str начиная с пощиции pos. Если такого нет, то вернет -1, иначе индекс где нашлось
  7. str.includes(substr, pos) - возвращает true, если в строке str есть подстрока substr, либо false, если нет
  8. str.startsWith()/str.endsWith()- проверяет, что строка начинается/заканчивается
  9. str.split(delim) - преобразует строку в массив с делителем delim

Методы массивов

*Массив это упорядочный объект

    Простые

  1. arr[0] - вернет первый элемент массива
  2. arr.length - вернет колличество элементов
  3. arr.at() = arr[], но в () можно указывать отрицательно и поиск будет с конца
  4. arr.push добавляет элемент в конец
  5. arr.shift удаляет элемент в начале и возвращает его
  6. arr.pop удаляет последний элемент и возвращает его
  7. arr.unshift добавляет элемент в начало
  8. Перебор for(let fruit of fruits){} - проходит по значениям
  9. Перебор for (let key in arr){} - - проходит по ключам

    10. Сложные

  10. arr.splice(start[, deleteCount, elem1, ..., elemN]) - Он изменяет arr начиная с индекса start: удаляет deleteCount элементов и затем вставляет elem1, ..., elemN на их место. Возвращает массив из удалённых элементов.
  11. arr.slice - Он возвращает новый массив, в который копирует все элементы с индекса start до end (не включая end).
  12. arr.concat(arg1, arg2...) - В результате – новый массив, включающий в себя элементы из arr, затем arg1, arg2 и так далее.
  13. arr.forEach(function(item, index, array) {}); - перебирает массив и что то делает с его элементами
  14. arr.indexOf(item, from) ищет item начиная с индекса from и возвращает номер индекса, на котором был найден искомый элемент, в противном случае -1.
  15. arr.includes(item, from) ищет item начиная с индекса from и возвращает true, если поиск успешен.
  16. arr.find(function(item, index, array) {
    // если true - возвращается текущий элемент и перебор прерывается
    // если все итерации оказались ложными, возвращается undefined
    });
    где item – очередной элемент. index – его индекс. array – сам массив
  17. arr.filter(function(item, index, array) {
    // если `true` -- элемент добавляется к results и перебор продолжается
    // возвращается пустой массив в случае, если ничего не найдено
    });
  18. arr.map(function(item, index, array) {
    // возвращается новое значение вместо элемента
    });
  19. arr.sort(); - ортирует массив на месте, меняя в нём порядок элементов
  20. arr.reverse меняет порядок элементов в arr на обратный
  21. arr.join(glue) - преобразует массив в строку с делителем glue
  22. arr.reduce(function(accumulator, item, index, array) {
    // ...
    }, [initial]);
    accumulator – результат предыдущего вызова этой функции, равен initial при первом вызове (если передан initial), item – очередной элемент массива, index – его позиция, array – сам массив.
    arr.reduceRight - тоже самое, только с права на лево
  23. Array.isArray([]) проверка, что это массив

Функции

Задаются 3 способами:

  • function myFunction() {} - видно до и после объявления
  • let myFunction = function () {} - видно только после объявления
  • () => {} - стрелочное объявление, выполняется сразу

Свойства:

  • length - содержит количество параметров функции в её объявлении
  • name - имя функции

Запланированые функции:

  1. setTimeout(func|code, [delay], [arg1], [arg2], ...) позволяет вызвать функцию один раз через определённый интервал времени
  2. setInterval(func|code, [delay], [arg1], [arg2], ...) позволяет вызывать функцию регулярно, повторяя вызов через определённый интервал времени
  3. clearTimeout(timerId); - очищает таймер. timerId - название переменной, которая создается при создание set*

Декоратор(Обертка в кеш) это обёртка вокруг функции, которая изменяет поведение последней.:
С одним аргументом

function cachingDecorator(func) { let cache = new Map(); return function(x) { if (cache.has(x)) { // если кеш содержит такой x, return cache.get(x); // читаем из него результат } let result = func(x); // иначе, вызываем функцию cache.set(x, result); // и кешируем (запоминаем) результат return result; }; }

С несколькими

function cachingDecorator(func, hash) { let cache = new Map(); return function() { let key = hash(arguments); // (*) if (cache.has(key)) { return cache.get(key); } let result = func.call(this, ...arguments); // (**) cache.set(key, result); return result; }; }

Объекты

Задаются так:
let user = {
name:'Some',
age: 15
}

Для обращения к параметру объекта user.name или user[name]

Функции, которые находятся в свойствах объекта, называются «методами».

Методы могут ссылаться на объект через this.

Проверка наличия - ?

  • obj?.prop – возвращает obj.prop если obj существует, в противном случае undefined.
  • obj?.[prop] – возвращает obj[prop] если obj существует, в противном случае undefined.
  • obj.method?.() – вызывает obj.method(), если obj.method существует, в противном случае возвращает undefined.
  • toJSON(){} - метод в объекте, который описывает, как объект преобразовывать в JSON

Функция-конструктор

Такие функции используются для создания объектов

Имя функции-конструктора должно начинаться с большой буквы.

Функция-конструктор должна выполняться только с помощью оператора "new".

Время

  1. new Date() - текущие время и дата
  2. new Date(milliseconds) - с временем, равным количеству миллисекунд (тысячная доля секунды), прошедших с 1 января 1970 года UTC+0
  3. new Date(datestring) - если аргумент всего один, и это строка, то из неё «прочитывается» дата
  4. new Date(year, month, date, hours, minutes, seconds, ms) - с заданными компонентами в местном часовом поясе

    5. Методы времени

  5. getFullYear() - получить год (4 цифры)
  6. getMonth() - получить месяц, от 0 до 11.
  7. getDate() - получить день месяца, от 1 до 31, что несколько противоречит названию метода.
  8. getHours(), getMinutes(), getSeconds(), getMilliseconds() - получить, соответственно, часы, минуты, секунды или миллисекунды.
  9. getDay() - вернуть день недели от 0 (воскресенье) до 6 (суббота).
  10. getTime() - для заданной даты возвращает таймстамп – количество миллисекунд, прошедших с 1 января 1970 года UTC+0.
  11. getTimezoneOffset() - возвращает разницу в минутах между UTC и местным часовым поясом
    12. setFullYear(year, [month], [date])
  12. setMonth(month, [date])
  13. setDate(date)
  14. setHours(hour, [min], [sec], [ms])
  15. setMinutes(min, [sec], [ms])
  16. setSeconds(sec, [ms])
  17. setMilliseconds(ms)
  18. setTime(milliseconds)
  19. Date.now() - возвращает текущую метку времени
  20. Date.parse(str) - считывает дату из строки и возвращает таймстамп

JSON

JSON (JavaScript Object Notation) – это формат данных, который имеет собственный независимый стандарт и библиотеки для большинства языков программирования.

  1. JSON.stringify для преобразования объектов в JSON
  2. JSON.parse для преобразования JSON обратно в объект.
  3. JSON.stringify(value, [replacer, space]) - value значение для кодирования, replacer массив свойств для кодирования или функция соответствия, space дополнительное пространство (отступы) для форматирования
  4. JSON.stringify(value, replacer, space) – это количество пробелов, используемых для удобного форматирования.
  5. toJSON(){} - метод в объекте, который описывает, как объект преобразовывать в JSON
  6. JSON.parse(str, [reviver]) - декодировать JSON-строку.str - JSON для преобразования в объект, reviver - необязательная функция, которая будет вызываться для каждой пары (ключ, значение) и может преобразовывать значение.

Промисы

Полезная информация

Синтаксис

let promise = new Promise(function(resolve, reject) {
// функция-исполнитель (executor)
// "певец"
});

Параметры

  1. function - будет вызвана сразу
  2. resolve(value) — вызовется, если работа завершилась успешно, с результатом value.
  3. reject(error) —вызовется, если произошла ошибка, error – объект ошибки.

Внутренние свойства

  1. state («состояние») — вначале "pending" («ожидание»), потом меняется на "fulfilled" («выполнено успешно») при вызове resolve или на "rejected" («выполнено с ошибкой») при вызове reject.
  2. result («результат») — вначале undefined, далее изменяется на value при вызове resolve(value) или на error при вызове reject(error)
