Основные понятия

  1. Компоненты - блоки, из которых формируется html
  2. JSX - язык-надстройка для JS
  3. State/состояние -
  4. prop/свойство - то, что пишется при рендере в пропах
  5. React Hook - позволяет менять состояния в компонентых, созданных через функцию, а не через класс

Правила

  1. Кастомные компоненты должны начинаться с БОЛЬШОЙ буквы
  2. Свойства передаются только дочерним
  3. Состояние меняется только в компоненте, где это состояние прописано
  4. Все React-компоненты должны вести себя как чистые функции в плане своих свойств.

Хуки(методы) жизненгого цикла

  1. componentDidMount() запускается после того, как вывод компонента отрисован
  2. componentWillUnmount() -метод вызывается, когда компонент удаляется из DOM
