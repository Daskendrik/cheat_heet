Единицы измерения

  1. Пиксели: px - это самая базовая, абсолютная и окончательная единица измерения.
  2. Относительно шрифта: em. 1em – текущий размер шрифта
  3. Проценты % - как и em – относительные единицы.
  4. Единица rem: смесь px и em - Единица rem задаёт размер относительно размера шрифта элемента html.
  5. vw – 1% ширины окна
  6. vh – 1% высоты окна
  7. vmin – наименьшее из (vw, vh), в IE9 обозначается vm
  8. vmax – наибольшее из (vw, vh)
  9. clamp(min,now, max) - мощный интрумент регулировки размера. Работает так: у нас есть динамический размер 1vh, он будет у нас в now. В параметры min и max мы вставляем минимальный и максимальный размер, то есть, когда now выходит за пределы, величина будет или min или max. Подробнее туть

Псевдоклассы

  1. :hover - наведение
  2. :active - активно
  3. :checked - находит любые элементы radio checkbox или option, которые выбраны или включены.
  4. :default - находит элемент формы, установленный по умолчанию для группы связанных элементов.
  5. :disabled - находит любой отключённый элемент
  6. :empty находит любой элемент, у которого нет потомков
  7. :enabled находит любой включённый элемент. Элемент включён, если его можно активировать (например, выбрать, нажать на него или ввести текст) или поставить фокус.
  8. :first используется с @-правилом @page, представляя первую страницу документа при печати.
  9. :first-child находит любой элемент, являющийся первым в своём родителе.
  10. :last-child находит любой элемент, являющийся последним в его родителе.
  11. :first-of-type находит первого потомка своего типа среди детей родителя.
  12. :last-of-type находит последнего потомка с заданным тегом в списке детей родительского элемента.
  13. :fullscreen соответствует элементу, который в данный момент находится в полноэкранном режиме.
  14. :focus применяется, когда элемент (такой как input формы) получает фокус.
  15. :in-range находит элементы, значение которых соответствует указанному для них диапазону ограничений.
  16. :out-of-range находит элементы, значение которых находится вне диапазона их ограничений.
  17. :invalid находит любые input или form элементы, контент которых не проходит валидацию (en-US), в соответствии с типом поля.
  18. :link позволяет вам выбирать ссылки внутри элементов
  19. :not(X) - функция, принимающая простой селектор X в качестве аргумента. Он находит элементы, не соответствующие селектору.
  20. :optional находит любые input элементы, у которых не установлен атрибут required
  21. :only-child находит любой элемент, являющийся единственным потомком родителя.
  22. :read-only находит элементы, недоступные для редактирования пользователем.
  23. :read-write находит элементы, доступные для редактирования пользователем, такие как текстовые поля.
  24. :required находит любые input элементы, имеющие атрибут required.
  25. :target представляет уникальный элемент (целевой элемент) с подходящим id URL-фрагментом.может использоваться для выделения части страницы, на которую была сделана ссылка из оглавления .
  26. :valid находит любые input или form элементы, контент которых валиден (en-US), в соответствии с типом поля.
  27. :visited позволяет вам выбирать ссылки, которые были посещены

Отдельно про display, а именно его значения

Инфа

  1. none – элемент не показывается
  2. block – блочные элементы располагаются один над другим, вертикально, стремится расшириться на всю доступную ширину
  3. inline - элементы располагаются на той же строке, последовательно. Ширина и высота элемента определяются по содержимому. Поменять их нельзя.
  4. inline-block – располагается в строке, размер устанавливается по содержимому, Элемент всегда прямоугольный, работают свойства width/height.
  5. table-* - для таблицы целиком table, для строки – table-row, для ячейки – table-cell. Это действительно полноценная таблица
    • table { display: table }
    • tr { display: table-row }
    • thead { display: table-header-group }
    • tbody { display: table-row-group }
    • tfoot { display: table-footer-group }
    • col { display: table-column }
    • colgroup { display: table-column-group }
    • td, th { display: table-cell }
    • caption { display: table-caption }
  6. list-item - добавляет к блоку с содержимым ещё и блок с номером(значком) списка

Отдельно про float и clear

Инфа
  1. Элемент позиционируется как обычно, а затем вынимается из документа потока и сдвигается влево (для left) или вправо (для right) до того как коснётся либо границы родителя, либо другого элемента с float
  2. Если пространства по горизонтали не хватает для того, чтобы вместить элемент, то он сдвигается вниз до тех пор, пока не начнёт помещаться.
  3. Другие непозиционированные блочные элементы без float ведут себя так, как будто элемента с float нет, так как он убран из потока.
  4. Строки (inline-элементы), напротив, «знают» о float и обтекают элемент по сторонам.
  5. Из-за того, что блок с float удалён из потока, родитель не выделяет под него места. Родителю надо добавить тоже float
  6. clear - Применение этого свойства сдвигает элемент вниз до тех пор, пока не закончатся float'ы слева/справа/с обеих сторон.

Анимация

Инфа Инфа 2

  1. Анимация создается с помощью @keyframes где пишется animation-name
  2. animation-name – имя анимации прописывается в селекторе того, что будет анимироваться
  3. animation-duration – определяет время, в течение которого должен пройти один цикл анимации
  4. transition-delay – задержка начала перехода
  5. animation-timing-function - Настраивает ускорение анимации
  6. animation-delay Настраивает задержку между временем загрузки элемента и временем начала анимации.
  7. animation-iteration-count Определяет количество повторений анимации; вы можете использовать значение infinite для бесконечного повторения анимации.
  8. animation-direction Даёт возможность при каждом повторе анимации идти по альтернативному пути, либо сбросить все значения и повторить анимацию.
  9. animation-fill-mode Настраивает значения, используемые анимацией, до и после исполнения.
  10. animation-play-state Позволяет приостановить и возобновить анимацию.
