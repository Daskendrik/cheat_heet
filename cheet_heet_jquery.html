Немного схематично

Как обращаться к DOM. Селекторы

  • $('.class').
  • $('#id').
  • $('tag').
  • $('tag tag').
  • $('tag tag:first').
  • $('tag:eq(n)'). - берем из кучи n-ое

Псевдоселекторы

  • $(':animated'); // Возвращает все элементы, для которых выполняется анимация в текущий момент
  • $(':contains(me)'); // Возвращает все элементы с текстом 'me'
  • $(':empty'); // Возвращает все элементы, у которых нет потомков или текста
  • $(':parent'); // Возвращает все элементы с потомками или текстом
  • $('li:even'); // Возвращает все элементы с четным индексом(в данном случае)
  • $('li:odd'); // Возвращает все элементы с нечетным индексом(в данном случае)
  • $(':header'); // Возвращает все h1 - h6.
  • $('li:gt(4)'); // Возвращает все элементы с индексом большим, чем заданный номер
  • $('li:lt(4)'); // Возвращает все элементы с индексом меньшим, чем заданный номер
  • $(':only-child'); // Возвращает все

Часто используемые методы

    Использовала

  • $().on('click',function (){}) - вешаем на что то какое либо событие
  • $().hasClass('class')
  • $().removeClass('class')
  • $().addClass('class')

    • Не использовала

  • $('Ошибка исправлена!').replaceAll('.error'); В данном примере все элементы с классом error будут заменены только что созданным span
  • $('.error').replaceWith('Ошибка исправлена!'); первый селектор будет заменен вторым
  • wrap() / wrapAll() / wrapInner() - данные функции облегчают процесс оборачивания элементов в другие элементы
  • $().after().text('Данный текст будеть иметь того же родителя, что и #child'));
  • $().before().text('То же самое, но текст вставляется перед #child'));
  • $().change(function (){}) - генерируется объектами текстовых полей, полей ввода текста, полями выбора, и происходит оно при изменении значения целевого элемента
  • $('div').hide().delay(2000).show(); // div будет оставаться скрытым в течении 2 секунд, перед тем, как показаться.
